# 📊 Task-7 Sales Data Analysis Using SQLite in Python

## 📌 Project Overview
This project demonstrates how to use SQLite inside Python to perform basic sales data analysis.  
The objective is to create a small SQLite database, run SQL queries to calculate total quantity and revenue, and visualize the results using a simple bar chart.

---

## 🎯 Objectives
- Create a small SQLite database (`sales_data.db`)
- Create a sales table
- Insert sample sales data
- Run SQL queries inside Python
- Display results using print statements
- Visualize revenue using a bar chart

---

## 🛠 Tools & Technologies Used
- Python
- SQLite (sqlite3)
- Pandas
- Matplotlib
- Jupyter Notebook

---

## 🗄 Database Details

### Table Name: `sales`

| Column Name | Data Type | Description |
|------------|-----------|------------|
| id | INTEGER | Primary Key (Auto Increment) |
| product | TEXT | Product Name |
| quantity | INTEGER | Quantity Sold |
| price | REAL | Price Per Unit |
