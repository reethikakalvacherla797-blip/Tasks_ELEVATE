# 🛍️ Retail Business Performance & Profitability Analysis

## 📌 Project Overview

This project analyzes transactional retail data to uncover profit-draining product categories, optimize inventory turnover, and understand seasonal product behavior.
The goal is to support data-driven business decisions that improve profitability and operational efficiency.

---

## 🎯 Objectives

* Identify low-profit and loss-making product categories
* Improve inventory turnover efficiency
* Analyze seasonal sales patterns
* Provide strategic recommendations for stock optimization

---

## 🧰 Tools & Technologies

* **SQL** – Data cleaning, transformation, and profit analysis
* **Python** – Data analysis and visualization

  * Pandas
  * NumPy
  * Seaborn
  * Matplotlib
* **Tableau** – Interactive dashboard and business insights

---

## Project Workflow:

1.Imported retail datasets into SQL and performed data cleaning by handling missing and null records
2.Wrote SQL queries to compute profit margins across categories and sub-categories
3.Used Python (Pandas) to analyze correlations between inventory holding days and profitability
4.Built an interactive Tableau dashboard with filters for region, product type, and seasonality
5.Generated strategic recommendations to reduce overstocking and improve slow-moving inventory performance

---

## 📊 Key Insights

* Certain sub-categories consistently generate low margins
* Longer inventory holding periods reduce profitability
* Seasonal demand strongly influences revenue spikes
* Regional preferences affect product performance
